# DwollaBees

This is a page speed monitoring tool for Dwolla.com.

## Pages we are monitoring
* http://dwolla.com
* https://www.dwolla.com/register

## What we are monitoring
- Google Page Speed
* score 
* numberResources
* totalRequestBytes
* numberStaticResources
* numberJsResources
* numberCssResources
* htmlResponseBytes
* imageResponseBytes
* javascriptResponseBytes
* otherResponseBytes
- GTMetrix
* pageLoadTime
